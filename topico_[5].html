Planos de Fundo CSS

As propriedades de plano de fundo do CSS são usadas para definir os efeitos de segundo plano para elementos.

Cor de fundo

A propriedade background-color especifica a cor de fundo de um elemento. A cor de fundo de uma página é configurada assim:

body { background-color: lightblue; }

Imagem de fundo

A propriedade background-image especifica uma imagem para usar como plano de fundo de um elemento. Por padrão, a imagem é repetida para cobrir todo o elemento. A imagem de fundo de uma página pode ser configurada assim:

body { background-image: url("paper.gif"); }
